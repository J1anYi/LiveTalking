# Requirements: LiveTalking RAG Knowledge Base (v1.0)

**Status:** ✅ SHIPPED 2026-05-12

## Functional Requirements

### FR-1: Knowledge Base Construction
- [x] **FR-1.1** 支持从本地文档（PDF、TXT、Markdown、Word）导入知识
- [x] **FR-1.2** 支持从数据库（SQLite）导入知识
- [x] **FR-1.3** 支持从 REST API 服务获取知识
- [x] **FR-1.4** 支持文档分块（chunking）配置
- [ ] **FR-1.5** 支持增量更新知识库 (部分实现)

### FR-2: Vector Embedding
- [x] **FR-2.1** 支持多种 Embedding 模型选择
- [x] **FR-2.2** 支持使用 DashScope Embedding API
- [ ] **FR-2.3** 支持本地 Embedding 模型（如 text2vec）(未实现)
- [x] **FR-2.4** 向量存储支持持久化

### FR-3: Retrieval
- [x] **FR-3.1** 支持语义相似度检索（top-k）
- [x] **FR-3.2** 检索延迟 < 500ms
- [x] **FR-3.3** 支持检索结果相关性评分
- [x] **FR-3.4** 支持元数据过滤

### FR-4: LLM Integration
- [x] **FR-4.1** 在 chat 模式自动注入知识库上下文
- [x] **FR-4.2** 保持现有流式输出机制
- [x] **FR-4.3** 支持多轮对话上下文管理
- [x] **FR-4.4** 兼容现有 echo 模式（无知识库检索）

### FR-5: Configuration
- [x] **FR-5.1** 支持通过 CLI 参数启用/禁用知识库
- [x] **FR-5.2** 支持配置知识库路径
- [x] **FR-5.3** 支持配置检索参数（top-k, threshold）
- [x] **FR-5.4** 支持环境变量配置

## Non-Functional Requirements

### NFR-1: Performance
- [x] 检索延迟 < 500ms（P95）
- [x] 向量索引构建支持增量更新
- [ ] 内存占用 < 2GB（100万向量）(未验证)

### NFR-2: Compatibility
- [x] Python 3.10+
- [x] 兼容现有 LiveTalking 架构
- [x] 支持所有现有传输协议（WebRTC/RTMP/VirtualCam）

### NFR-3: Maintainability
- [x] 代码符合现有项目规范
- [x] 添加必要日志记录
- [x] 提供使用文档
