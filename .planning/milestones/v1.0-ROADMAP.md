## Milestone Summary

**Key Decisions:**
- 向量数据库: ChromaDB (轻量级，易于集成，支持本地持久化)
- Embedding 模型: DashScope text-embedding-v3 (1024 维度)
- 文档分块: 固定长度 + 重叠 (chunk_size=800, overlap=100)
- 配置优先级: CLI > 环境变量 > YAML 文件 > 默认值

**Issues Resolved:**
- Windows ChromaDB 文件锁定问题 (使用 tempfile.mkdtemp + 手动清理)
- DashScopeEmbedding mock 测试问题 (直接在实例上设置 MagicMock)

**Technical Debt:**
- 预存在的 test_vector_store.py 和 test_integration.py 测试失败 (Windows 文件锁定)
