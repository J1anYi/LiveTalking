  <action>
    创建 rag/sources/registry.py：
    
    1. SourceRegistry 类：
       ```python
       class SourceRegistry:
           def __init__(self):
               self._loaders: dict[str, DocumentLoader] = {}
           
           def register(self, name: str, loader: DocumentLoader) -> None:
               self._loaders[name] = loader
           
           def get(self, name: str) -> DocumentLoader | None:
               return self._loaders.get(name)
           
           def load_all(self) -> list[tuple[str, dict]]:
               # 加载所有注册的数据源
               results = []
               for name, loader in self._loaders.items():
                   results.extend(loader.load())
               return results
       ```
    
    2. create_loader_from_config(config: SourceConfig) -> DocumentLoader 工厂函数：
       - 根据 config.type 创建对应加载器
       - file -> FileLoader
       - sqlite -> SQLiteConnector
       - api -> APILoader
    
    per D-12: 数据源注册表支持动态加载
