# Phase 2 Plan 03: 文档处理器

**目标:** 实现文档加载和分块处理器，支持中文优化分块。

## 背景

根据 Phase 1 验证结果:
- chunk_size: 800
- chunk_overlap: 100
- 中文分隔符: ["\n\n", "\n", "。", "！", "？", "；", "，", " "]
- 使用 langchain-text-splitters 1.1.2

## 任务

### Task 1: 实现 DocumentProcessor 类

- rag/API_DESIGN.md - 查看完整 API 设计
- .planning/phases/01-research-design/01-VERIFICATION.md - 查看验证结果
</read_first>

<action>
创建 `rag/document_processor.py` 文件:

1. 定义中文分隔符常量:
   ```python
   CHINESE_SEPARATORS = ["\n\n", "\n", "。", "！", "？", "；", "，", " "]
   ```

2. 实现 `__init__` 方法:
   - 参数: chunk_size=800, chunk_overlap=100, separators=None
   - 如果 separators 为 None，使用 CHINESE_SEPARATORS
   - 创建 RecursiveCharacterTextSplitter 实例

3. 实现 `process_text(text, metadata=None) -> list[dict]` 方法:
   - 调用 splitter.split_text() 分割文本
   - 返回 [{"text": chunk, "metadata": metadata or {}}, ...]

4. 实现 `process_file(file_path) -> list[dict]` 方法:
   - 检测文件扩展名
   - 支持 .txt 和 .md（直接读取）
   - 支持 .pdf（使用 pypdf）
   - 提取文件元数据（source, type）
   - 调用 process_text() 处理

5. 实现 `process_directory(dir_path, recursive=True) -> list[dict]` 方法:
   - 遍历目录
   - 对每个支持的文件调用 process_file()
   - 合并所有结果
</action>
