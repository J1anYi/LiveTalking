# Phase 2 Plan 01: DashScope Embedding 客户端

**目标:** 实现 DashScope Embedding API 客户端，支持文本向量生成。

## 背景

根据 Phase 1 验证结果:
- Embedding 模型: text-embedding-v3
- 向量维度: 1024
- API 基础 URL: https://dashscope.aliyuncs.com/compatible-mode/v1

## 任务

### Task 1: 实现 DashScopeEmbedding 类

<read_first>
- rag/__init__.py - 查看现有 Protocol 定义
- rag/API_DESIGN.md - 查看完整 API 设计
- .planning/phases/01-research-design/01-VERIFICATION.md - 查看验证结果
</read_first>

<action>
创建 `rag/embeddings.py` 文件，实现 DashScopeEmbedding 类:

1. 实现 `__init__` 方法:
   - 参数: model="text-embedding-v3", api_key=None, base_url, timeout=30.0
   - 从环境变量 DASHSCOPE_API_KEY 获取 API Key
   - 创建 OpenAI 客户端实例

2. 实现 `embed(texts: list[str]) -> list[list[float]]` 方法:
   - 批量生成向量，最多支持 20 个文本
   - 调用 client.embeddings.create() API
   - 返回向量列表（每个向量 1024 维）

3. 实现 `embed_query(query: str) -> list[float]` 方法:
   - 单个查询向量生成
   - 调用 embed() 方法并返回第一个结果

4. 实现 `dimensions` 属性:
   - 返回 1024

5. 错误处理:
   - ValueError: 参数无效（空文本、超过 20 个）
   - RuntimeError: API 调用失败
</action>

- 文件包含 `def embed(self, texts: list[str]) -> list[list[float]]:`
- 文件包含 `def embed_query(self, query: str) -> list[float]:`
- 文件包含 `@property def dimensions(self) -> int: return 1024`
- 导入语句包含 `from openai import OpenAI`
</acceptance_criteria>
