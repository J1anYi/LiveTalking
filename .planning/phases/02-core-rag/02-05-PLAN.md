# Phase 2 Plan 05: 文件加载器

**目标:** 实现本地文件加载器，支持多种文档格式。

## 背景

支持的文件格式:
- .txt - 纯文本
- .md - Markdown
- .pdf - PDF 文档（需要 pypdf）
- .docx - Word 文档（需要 python-docx，可选）

### Task 2: 实现 FileLoader 类

<read_first>
- rag/loaders/__init__.py - 刚创建的包
- rag/API_DESIGN.md - 查看完整 API 设计
</read_first>

<action>
创建 `rag/loaders/file_loader.py` 文件:

1. 定义默认支持的扩展名:
   ```python
   DEFAULT_EXTENSIONS = [".txt", ".md", ".pdf"]
   ```

2. 实现 `__init__` 方法:
   - 参数: supported_extensions=None, encoding="utf-8"
   - 如果 supported_extensions 为 None，使用 DEFAULT_EXTENSIONS

3. 实现 `load(file_path) -> tuple[str, dict]` 方法:
   - 检测文件扩展名
   - 根据扩展名选择加载方法
   - 返回 (content, metadata)
   - metadata 包含: source, type, size

4. 实现 `_load_text(file_path)` 方法:
   - 使用指定编码读取文件
   - 返回文本内容

5. 实现 `_load_pdf(file_path)` 方法:
   - 使用 pypdf.PdfReader
   - 提取所有页面文本
   - 合并为单个字符串

6. 实现 `load_directory(dir_path, recursive=True) -> list[tuple[str, dict]]` 方法:
   - 遍历目录
   - 对每个支持的文件调用 load()
   - 返回所有结果列表

7. 错误处理:
   - 不支持的扩展名抛出 ValueError
   - 文件不存在抛出 FileNotFoundError
   - 编码错误抛出 UnicodeDecodeError
</action>

- 文件包含 `def _load_text(self, file_path)`
