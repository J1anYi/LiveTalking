2. 实现 `get_default_rag_config() -> dict` 函数：返回 DEFAULT_RAG_CONFIG 的副本

3. 实现 `load_rag_config(filepath: str | None = None) -> dict` 函数：
   - 默认 filepath 为 "data/rag_config.yaml"
   - 使用 yaml.safe_load() 加载
   - 文件不存在时返回空字典 {}
   - 从 YAML 的 "rag:" 键下读取配置
   - 返回合并后的配置（默认值 + 文件值）

4. 实现 `save_rag_config(config: dict, filepath: str) -> None` 函数：
   - 将配置保存到 YAML 文件
   - 使用 yaml.dump()，格式化输出
